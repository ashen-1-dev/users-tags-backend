

## Требования
Node.js 16.x<br> 
npm 8.x

## Установка

```bash
$ npm install
```



## Настройка окружения

В зависимости от окружения необходима создать и заполнить файл ```.env.development``` или ```.env.production```
Пример можно скопировать с файла ```.env.example```

## Запуск

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
